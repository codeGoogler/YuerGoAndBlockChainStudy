**cal命令**用于显示当前日历，或者指定日期的日历。

## 语法

`cal(选项)(参数)`

## 选项

```
-l：显示单月输出；
-3：显示临近三个月的日历；
-s：将星期日作为月的第一天；
-m：将星期一作为月的第一天；
-j：显示“julian”日期；
-y：显示当前年的日历。
```

## 参数

```
月：指定月份；
年：指定年份。
```

## 常用实例

1）显示当前月份日历

```
# cal

25 26 27 28 29 30 31
```
